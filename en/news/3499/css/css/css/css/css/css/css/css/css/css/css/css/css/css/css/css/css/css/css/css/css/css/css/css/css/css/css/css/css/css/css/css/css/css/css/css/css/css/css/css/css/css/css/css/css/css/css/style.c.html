12.06.2019 ECOINSTRUMENT & Foundation “Clean Hands Clean Rivers”
ECOINSTRUMENT  LLC. together with the Foundation “Clean Hands Clean Rivers” at a special press conference held on June 11, 2019 in Rostov-on-Don presented to the public and the media the results of the expedition along the Don River.
During the expedition in May-June 2019, the Foundation’s specialists for the first time in Russia carried out a quantitative assessment of microplastic content in river water, as well as carried out environmental monitoring of the Tsimlyansk reservoir.
